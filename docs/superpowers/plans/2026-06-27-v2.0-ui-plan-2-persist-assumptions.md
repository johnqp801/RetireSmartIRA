# V2.0 UI — Plan 2: Persist MultiYearAssumptions — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist `MultiYearAssumptions` across app launches so dismissed banners (and every other Multi-Year assumption) survive a relaunch, using the app's existing UserDefaults persistence pattern.

**Architecture:** `MultiYearAssumptions` (already `Codable` with a defensive `init(from:)`) becomes a stored property on `DataManager` and is saved/loaded as one JSON blob via `PersistenceManager` (the same pattern as `iraAccounts` / `completedActionKeys`). `MultiYearStrategyManager` seeds its `assumptions` from `DataManager` on `attach` and mirrors changes back via `didSet`; `MultiYearPlanView` triggers `dataManager.saveAllData()` on change (matching the `SettingsView` `.onChange` save pattern).

**Tech Stack:** Swift, SwiftUI, Swift Testing. Native macOS + iOS, no Catalyst.

## Global Constraints

- Additive only: do NOT modify, replace, or remove the Scenarios or Tax Summary tabs or any single-year view.
- No engine changes: this plan touches persistence + manager wiring only. No optimizer / projection / tax-loop changes.
- No em dash character (U+2014) in any user-facing copy.
- Persistence MUST go through the existing `PersistenceManager` + `DataManager.saveAllData()` chokepoint and honor the injectable `defaults:` parameter (so the DEBUG `DemoProfile.defaults` suite routing keeps working).
- TEST SAFETY: never call `DataManager.saveAllData()` in a test (it writes the real `DemoProfile.defaults` suite in DEBUG and would overwrite the manual-test demo profile). Test persistence only through `PersistenceManager.saveAll/loadAll(... , defaults:)` with an ephemeral `UserDefaults(suiteName:)`.
- `dismissedInsightKeys` is pure UI state (the engine ignores it). Persisting it MUST NOT trigger a recompute: do not add `dataManager.multiYearAssumptions` to `observeUpstreamChanges()`.

```swift
import Testing
import Foundation
@testable import RetireSmartIRA

@Suite("Persist MultiYearAssumptions", .serialized)
@MainActor
struct PersistMultiYearAssumptionsTests {

    private func ephemeralSuite() -> UserDefaults {
        UserDefaults(suiteName: "test-mya-\(UUID().uuidString)")!
    }

    @Test("assumptions round-trip through PersistenceManager")
    func roundTrips() {
        let suite = ephemeralSuite()
        let dm = DataManager(skipPersistence: true)
        dm.multiYearAssumptions.dismissedInsightKeys = ["survivor", "ssNudge"]
        dm.multiYearAssumptions.currentTaxableBalance = 250_000
        dm.multiYearAssumptions.terminalLiquidationTaxRate = 0.30
        PersistenceManager.saveAll(from: dm, defaults: suite)

        let reloaded = DataManager(skipPersistence: true)
        PersistenceManager.loadAll(into: reloaded, defaults: suite)
        #expect(reloaded.multiYearAssumptions.dismissedInsightKeys == ["survivor", "ssNudge"])
        #expect(reloaded.multiYearAssumptions.currentTaxableBalance == 250_000)
        #expect(reloaded.multiYearAssumptions.terminalLiquidationTaxRate == 0.30)
    }

    @Test("missing key leaves default assumptions")
    func missingKeyDefaults() {
        let suite = ephemeralSuite()
        let dm = DataManager(skipPersistence: true)
        PersistenceManager.loadAll(into: dm, defaults: suite)  // empty suite
        #expect(dm.multiYearAssumptions.dismissedInsightKeys.isEmpty)
        #expect(dm.multiYearAssumptions == MultiYearAssumptions())
    }
}
```
