# V2.1.0 Phase 1b — Charitable-Giving Input Model — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the multi-year charitable-giving input model — `GivingIntent` (fixed amount / % of RMD) separated from the QCD funding method (`qcdFirst` / `fixedQCD`) — to `MultiYearStaticInputs`, seeded by the adapter from the single-year giving entries, so the recurring-QCD *application* task (Phase 1c) can consume it.

**Architecture:** Model intent and funding as two small value types wrapped in a `CharitableGivingPlan`, carried on `MultiYearStaticInputs` (default `.none`). The adapter seeds a default plan from the single-year `scenarioTotalCharitable`. This phase is purely additive: nothing in `ProjectionEngine` consumes the plan yet (exactly like the existing `year1PrimaryQCD`/`year1SpouseQCD` fields are carried but not yet applied). Phase 1c wires it into the engine. No numeric behavior changes here.

**Tech Stack:** Swift, Swift `Testing` framework (`import Testing`, `@Test`, `@Suite`, `#expect`, `@MainActor`), `@testable import RetireSmartIRA`. Native macOS/iOS Xcode project (new `.swift` files auto-compile via `PBXFileSystemSynchronizedRootGroup` — no pbxproj edit needed).

## Global Constraints

- **Additive only — no behavior change.** The full suite (1,713 cases) must stay green with no rebaselines. The new `charitableGivingPlan` field is carried but consumed by nothing in this phase (Phase 1c consumes it); it must default to `.none` so every existing `MultiYearStaticInputs(...)` call site compiles and behaves unchanged.
- **Intent vs funding are separate axes** (spec §6 Phase 1): `GivingIntent` is *how much* to give; `QCDFundingMethod` is *how* to fund it. Do not conflate them.
- `GivingIntent.percentOfRMD` stores a **fraction** (0.25 = 25%), interpreted against the household total RMD (all traditional sources) in Phase 1c.
- Adapter default: `intent = .fixedAnnualAmount(dataManager.scenarioTotalCharitable)`, `funding = .qcdFirst`, `maintainRealValue = true`. `scenarioTotalCharitable` ([DataManager.swift:2662]) already = QCD + cash charitable (the household total giving).
- **Authoritative test counts:** `xcodebuild test` stdout undercounts large Swift-Testing suites — use `-resultBundlePath <dir>` + `xcrun xcresulttool` for the full-suite count.
- Tests are the source of truth.

```swift
//
//  CharitableGiving.swift
//  RetireSmartIRA
//
//  The multi-year charitable-giving input model. Intent (HOW MUCH to give) is kept
//  separate from the funding method (HOW to fund it — cash vs QCD), so the two can vary
//  independently and QCD sourcing is decided per year in the engine (Phase 1c). This file
//  is inputs only; the engine does not consume it yet.
//

import Foundation

/// How much the household intends to give each year.
enum GivingIntent: Equatable, Sendable {
    /// A fixed dollar target (see `CharitableGivingPlan.maintainRealValue` for inflation).
    case fixedAnnualAmount(Double)
    /// A fraction of the household total RMD (0.25 == 25%). Only bites at RMD age; interpreted
    /// against the combined RMD across all traditional sources in Phase 1c.
    case percentOfRMD(Double)
}

/// How the giving target is funded through QCDs (the remainder is cash charitable).
enum QCDFundingMethod: Equatable, Sendable {
    /// Fund the target with QCDs to the extent each spouse is eligible (the default).
    case qcdFirst
    /// Route a fixed dollar amount through QCDs; the remainder of the target is cash.
    case fixedQCD(Double)
}

/// The household's recurring charitable-giving plan for the multi-year projection.
struct CharitableGivingPlan: Equatable, Sendable {
    var intent: GivingIntent
    var funding: QCDFundingMethod
    /// Fixed-amount intent only: hold the target constant in real terms (inflate the nominal
    /// target with CPI) rather than flat-nominal. Ignored for `.percentOfRMD`.
    var maintainRealValue: Bool

    /// A plan directing no charitable giving.
    static let none = CharitableGivingPlan(intent: .fixedAnnualAmount(0), funding: .qcdFirst, maintainRealValue: true)

    /// True when the plan directs any giving.
    var hasGiving: Bool {
        switch intent {
        case .fixedAnnualAmount(let amount): return amount > 0
        case .percentOfRMD(let fraction): return fraction > 0
        }
    }
}
```
