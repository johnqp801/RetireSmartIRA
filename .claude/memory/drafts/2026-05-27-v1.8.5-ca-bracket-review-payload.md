**Mental Health Services Tax (MHST):**
- 1.00% additional on taxable income > $1,000,000 (statutory threshold, NOT inflation-indexed)
- Same $1M threshold for Single, MFJ, HoH per CA tax law (RTC §17043)
- Top effective rate: 12.30% + 1.00% = 13.30%
- For MFJ specifically: 1% MHST kicks in at $1,000,000 (inside the 11.3% bracket which runs $891,542 to $1,485,906), so the engine models this with an extra bracket boundary at $1M to switch from 11.3% to 12.3% (effective with MHST).

### Bracket-engine semantics (so you can interpret the code correctly)

The `B(threshold: X, rate: Y)` array is a list of bracket-start points. Income in the range [X_n, X_{n+1}) is taxed at rate Y_n. The engine integrates across brackets to compute total tax on a given income.

For the MFJ schedule, note the model inserts an artificial bracket boundary at $1,000,000 (between $891,542 and $1,485,906) so the MHST surtax (which kicks in at $1M for all filers per RTC §17043) is correctly applied in the $1M-$1,485,906 zone. So the visible "12.3%" at threshold $1M is actually "11.3% base + 1% MHST", and the visible "13.3%" at $1,485,906 is "12.3% base + 1% MHST". This is the engine's modeling convention for combining the regular schedule with the MHST surtax in a single bracket array.
